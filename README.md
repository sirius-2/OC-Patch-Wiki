### 直接启动到macOS
+ `Ctrl + 数字`选择默认启动入口  
+ `Misc -> Boot -> ShowPicker -> NO`  
> 默认入口选择要求`Misc -> Security -> AllowSetDefault -> YES`，设置完成后可为`NO`
